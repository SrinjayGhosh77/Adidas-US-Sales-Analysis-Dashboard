# Adidas-US-Sales-Analysis-Dashboard

## Project Overview

This project is an interactive Microsoft Excel dashboard built using the Adidas US Sales dataset. The dashboard provides valuable business insights by analyzing sales performance, product performance, retailer performance, regional sales, and Year-over-Year growth. Data was cleaned and prepared in Excel, while Pivot Tables, Pivot Charts, KPI Cards, and Slicers were used to create an interactive dashboard.

## Business Questions Answered

### Sales Performance

* What is the overall Revenue, Profit, Profit Margin, and Units Sold?
* Which year generated the highest sales?
* Which month recorded the highest sales?
* How does Revenue compare with Profit over time?
* Which months show the highest and lowest sales performance?
* Are sales increasing because of higher unit sales or higher prices?

### Sales Channel Analysis

* Which sales channel generates the highest revenue?
* Which sales channel is the most profitable?
* Which sales channel performs best over time?

### Product Analysis

* Which product sells the most units?
* Which product generates the highest revenue?
* Which product has the highest profit margin?
* Which products have high sales but relatively low profit?
* Are premium-priced products more profitable?

### Retailer Analysis

* Which retailer generates the highest sales?
* Which retailer contributes the highest profit?

### Regional Analysis

* Which region generates the highest sales?
* Which state contributes the highest revenue?
* Which states are underperforming?
* Which region has the highest Year-over-Year (YoY) growth?
